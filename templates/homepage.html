About The Genomic Ascertainment Cohort (TGAC) Browser Beta

TGAC is a data resource that aims to improve our understanding of the phenotypic consequence of genetic variation. Exome and genome data from patients in contributing cohorts are available to view in aggregate via this browser. This project is designed to enable further study of individuals via genomic ascertainment without prior knowledge of phenotype.

The initial dataset included in this beta release of the TGAC Browser comprises exomes from 938 participants in the NHGRI-sponsored ClinSeq® study. We plan to add more exome and genome data from other investigators soon. See “About The Genomic Ascertainment Cohort” for information about this effort. This project is a collaboration between NHGRI (Les Biesecker) and NIAMS (Richard Siegel).

All data here are released under a Fort Lauderdale Agreement for the benefit of the IRP community. Please read and understand that agreement before using the data.

Our team will facilitate the use of TGAC for your proposed project, including the process of contacting the individuals identified as having variants of interest. If you have a research project that you would like to propose, please complete the project proposal form.

NIH gratefully acknowledges Daniel MacArthur and the Broad Institute for sharing the gnomAD web architecture with us to support this pilot.
