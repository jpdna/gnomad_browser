The Genome Ascertainment Cohort (TGAC) Browser Beta

Examples - Gene: PCSK9, Variant: 1-55505647-G-T

The Genome Ascertainment Cohort (TGAC) is a data resource that aims to improve our understanding of the phenotypic consequence of genetic variation. The project is designed to enable further study of individuals via genomic ascertainment without prior knowledge of phenotype. TGAC is a collaboration between NHGRI (Les Biesecker ) and NIAMS (Richard Siegel ).

Investigators may use this web interface to view aggregate exome and genome data from patients in contributing cohorts. Patients in TGAC have agreed to be re-contacted for secondary phenotyping studies. After identifying a variant or variants of interest, investigators may propose projects to study phenotypes that may be associated with the genotype. The initial dataset included in this beta release of the TGAC Browser comprises exomes from 938 participants in the NHGRI-sponsored ClinSeq® study. We plan to add more exome and genome data from other investigators soon. This beta release does not contain variants in genes on the X or Y chromosome, but these data will be added soon. See “About The Genomic Ascertainment Cohort” for more information.

All data here are released under a Fort Lauderdale Agreement for the benefit of the IRP community. Please read and understand that agreement before browsing the variants.

We thank Konrad Karczewski, Daniel MacArthur, and the gnomAD browser team for the original gnomAD browser code modified to create this resource.
